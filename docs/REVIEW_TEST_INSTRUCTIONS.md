# Chrome Web Store Review Test Instructions

Use this in the dashboard's test instructions field if reviewers need guidance.

1. Install the submitted extension package.
2. Open one of the supported AI chat sites, such as `https://chatgpt.com/`.
3. Open or create a conversation with several messages.
4. Confirm the MessageRail sidebar appears on the right side of the page.
5. Click a sidebar row and confirm the page scrolls to that message.
6. Type into the MessageRail search field and confirm the sidebar filters results.
7. Click the sidebar collapse button and confirm the right-edge rail tab remains.
8. Click the rail tab and confirm the sidebar opens again.
9. Optional: use `Alt+M` to toggle the sidebar and `Alt+Shift+M` to focus search.

No test account is provided. The extension only operates on supported AI chat pages and does not make network requests.
