# MessageRail Privacy Policy

Effective date: April 29, 2026

MessageRail is a privacy-first browser extension that adds a local message index sidebar to supported AI chat applications.

## Summary

MessageRail does not collect, sell, transmit, or share user data. MessageRail does not use telemetry, analytics, advertising, tracking pixels, remote scripts, or network requests.

## Data Processed Locally

MessageRail reads message text from supported AI chat pages in order to build an on-page message index, search results, and jump targets. This processing happens locally in the user's browser.

MessageRail may store local preferences, such as sidebar collapsed state, using `chrome.storage.local`. MessageRail also includes local IndexedDB storage infrastructure for message pin data.

## Data Collection

MessageRail does not collect user data from the browser and does not send user data to any server.

## Data Sharing

MessageRail does not sell, rent, transfer, or share user data with third parties.

## Network Requests

MessageRail does not make network requests. It does not use remote code, remote fonts, remote images, telemetry endpoints, analytics services, or advertising services.

## Permissions

MessageRail requests `storage` permission to save local extension preferences.

MessageRail requests host permissions only for supported AI chat sites so it can inject the sidebar and index the current conversation locally.

## Data Retention And Deletion

Local extension data remains in the user's browser until the user clears extension/site data or uninstalls the extension.
